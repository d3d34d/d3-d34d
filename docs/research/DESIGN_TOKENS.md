# Design Tokens

## Colors

| Token | Value | usage |
| :--- | :--- | :--- |
| --background | #030405 | Main page background |
| --foreground | #FFFFFF | Default text color |
| --primary | #00FF9C | Terminal green accent |
| --secondary | #00E3FF | Terminal blue accent |
| --border | #00FF9C | Card and input borders |
| --muted | #A1A1AA | Secondary text |
| --card-bg | #0B0E0E | Section background cards |

### Extended Palette
- **Yellow Tag**: #FFB800
- **Purple Tag**: { bg: #2B1D33, text: #D18BFF }
- **Blue Tag**: { bg: #1D2433, text: #8BBDFF }

## Typography

- **Font Family**: `JetBrains Mono`, monospace
- **Weights**: 400 (Regular), 700 (Bold)

### Type Scale
- **Nav**: 14px
- **Body**: 16px
- **Heading Large**: 32px
- **Stats**: 24px

## Spacing & Layout
- **Page Max-Width**: 1440px
- **Border Radius**: 0px (Hard edges for terminal look)
- **Grid Gap**: 24px

## Visual Effects
- **Scanlines**: Subtle horizontal overlay
- **Dot Pattern**: Background grid dot pattern (optional)
- **Glow**: Subtle neon green outer glow on primary elements
